# python-calender

## 概要
Python（tkinter）で作成したシンプルなカレンダーアプリケーションです。月ごとのカレンダー表示や前月・次月への移動が可能です。

## スクリーンショット / デモ
スクリーンショットやデモGIFは提供していません。

## 必要なライブラリ
- Python 3.8以上
- tkinter（標準ライブラリ）
- calendar（標準ライブラリ）
- datetime（標準ライブラリ）

## セットアップ方法
本アプリは標準ライブラリのみで動作するため、追加のパッケージインストールは不要です。

## 実行方法
```bash
python calendar_app.py
```
